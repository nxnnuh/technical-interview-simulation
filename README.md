# technical-interview-simulation
This project is a technical interview simulation where I solved an algorithmic problem while explaining my thought process in real time. The goal was to demonstrate problem-solving, communication, and coding skills under interview conditions.
The problem involved finding the unique common elements between two arrays. I implemented an optimized solution using sets to efficiently compute the intersection.

## Problem Description
Given two integer arrays, return a list of unique elements that appear in both arrays.

Example:
Input: nums1 = [1,2,2,1], nums2 = [2,2]
Output: [2]

## Approach
I used a set-based approach to remove duplicates and perform fast lookups. By converting both arrays into sets and computing their intersection, I was able to reduce time complexity compared to a brute force solution.

## Technologies Used
- Python
- Data Structures (Sets)
- Algorithm Optimization

## What I Learned
This project helped me improve my ability to explain my thinking while solving problems. I learned the importance of discussing multiple approaches, especially starting with a brute force solution before optimizing.
